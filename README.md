# Innook - 电影级专注学习

> Cinematic Focus - 每一帧都像精心拍摄的电影截图

## 📱 多设备数据同步

通过**导出/导入文件**在 iPad 和电脑间同步数据，无需联网！

### 使用方法

**设备 A（导出数据）：**
1. 打开网页，点击左上角「导出备份文件」
2. 下载 JSON 文件
3. 通过微信/QQ/AirDrop 发送到设备 B

**设备 B（导入数据）：**
1. 打开网页，点击左上角「导入备份文件」
2. 选择收到的 JSON 文件
3. 数据自动恢复！

**快捷方式（复制粘贴）：**
1. 设备 A：点击「复制数据文本」
2. 发送文本到设备 B（微信/QQ）
3. 设备 B：点击「导入备份文件」→ 粘贴文本

## ✨ 核心特性

### 🎬 双态界面设计

**准备模式**
- 自定义计时器（1-180 分钟）
- 场景选择（4 张背景图）
- 声音氛围（篝火/雨声/森林/海浪）

**专注模式**
- 全屏电影级背景
- 巨大倒计时数字
- 随机激励金句

### 📊 数据分析

- 日历热力图
- 每日详情
- 任务分布图

### ✅ 诚实打卡

- 询问是否专注
- 记录学习内容
- 真实时长统计
