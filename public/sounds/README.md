# FlowSpace 音频资源目录

## 音频文件清单

将以下 `.mp3` 文件放置在本目录：

| 文件名 | 描述 | 推荐时长 |
|--------|------|----------|
| `fire.mp3` | 🔥 篝火燃烧声 | 5-10 分钟循环 |
| `rain.mp3` | 🌧️ 雨声 | 5-10 分钟循环 |
| `forest.mp3` | 🌲 森林鸟鸣 | 5-10 分钟循环 |
| `cafe.mp3` | ☕ 咖啡厅环境音 | 5-10 分钟循环 |
| `brown-noise.mp3` | 🎵 棕噪音 | 5-10 分钟循环 |
| `timer-end.mp3` | ⏰ 计时结束提示音 | 3-5 秒 |

## 格式要求

- **格式**: MP3 (兼容性最佳)
- **采样率**: 44.1kHz 或 48kHz
- **比特率**: 128-192 kbps (平衡质量与文件大小)
- **声道**: 立体声 (环境音) / 单声道 (提示音)

## 文件大小建议

- 环境音: 1-5 MB (5分钟循环)
- 提示音: 50-200 KB

## 临时占位方案

如果暂时没有音频文件，应用会优雅降级：
- 音频加载失败时显示静音状态
- 不会影响其他功能的正常使用
