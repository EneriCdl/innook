# DeepFocus 音频资源

## 必需音频文件

将以下 `.mp3` 文件放置在本目录：

| 文件名 | 描述 | 推荐来源 |
|--------|------|----------|
| `fire.mp3` | 🔥 篝火燃烧声 | [Freesound #506037](https://freesound.org/s/506037/) (CC0) |
| `rain.mp3` | 🌧️ 雨声 | [Freesound #531851](https://freesound.org/s/531851/) (CC0) |
| `forest.mp3` | 🌲 森林鸟鸣 | [Freesound #460668](https://freesound.org/s/460668/) (CC0) |
| `ocean.mp3` | 🌊 海浪声 | [Freesound #460668](https://freesound.org/s/460668/) (CC0) |

## 格式要求

- **格式**: MP3
- **采样率**: 44.1kHz
- **比特率**: 128-192 kbps
- **时长**: 5-10 分钟循环

## 临时方案

如果没有音频文件，应用会优雅降级：
- 音频加载失败时显示提示
- 其他功能正常运行
